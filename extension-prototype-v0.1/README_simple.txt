Password Manager Extension v0.2 (Simple UI)

Features:
- Autofill Detection toggle (UI only)
- Status display (manual, placeholder)
- Password Vault: list, add, delete, reveal, open
- Add Credential form (site, username/email, password, status)
- Export Logs button
- No advanced/fancy UI, just basic HTML/CSS

How to use:
- Load popup_simple.html as your extension popup
- Use the Add Password button to add credentials
- Use the Password Vault to view, reveal, open, or delete credentials
- Use Export Logs to download a placeholder log file
